# vue4-cli

cli 폴더 만들기 ㄱ

하고 bash에서 

```
npm i -g @vue/cli
```

```
vue create todo-vue-cli
```

디폴트 선택 ㄱㄱ

```
cd todo-vue-cli
```

```
npm run serve
```

서버 끄고

```
touch src/components/TodoList.vue
```

그 src/components에 TodoList.vue 만들어서 웹팩에 만들었던거 복붙 ㄱ

App.vue에서도 템플릿이랑 스크립 복붙 ㄱ

다시 서버 실행 ㄱ

끝임
